  - 🎨 3D modellashtirish (Blender, Unity, Unreal Engine)

- **Aqlli javob tizimi** - Kalit so'zlar asosida avtomatik javoblar
- **Mavzu filtrlash** - Faqat kerakli yo'nalish bo'yicha suhbat

- **Tezkor javoblar** - Kod namunalari va amaliy maslahatlar
